# Manual de Usuario

## 1. Introduccion
Sevillaneando es una aplicacion movil para descubrir eventos, guardar recomendaciones y participar en chats asociados a cada evento.

## 2. Requisitos para usar la app
- Dispositivo Android o iOS compatible con Expo/React Native.
- Conexion a Internet.
- Cuenta de usuario para acceder a funcionalidades privadas (chat, perfil, eventos propios).

## 3. Primer acceso
1. Abre la app.
2. Registra una cuenta o inicia sesion.
3. Completa tu perfil (nombre, foto y preferencias de categorias).

## 4. Pantalla principal
Desde la pantalla principal puedes:
- Ver eventos recomendados.
- Buscar eventos por texto o filtros.
- Abrir el detalle de un evento.
- Navegar a secciones de perfil y ajustes.

## 5. Eventos
### 5.1 Ver detalle de evento
En el detalle de un evento puedes:
- Consultar informacion general (titulo, fecha, ubicacion, descripcion).
- Visualizar el evento en mapa.
- Compartir enlace del evento.

### 5.2 Crear evento
1. Entra en la opcion de crear evento.
2. Rellena titulo, descripcion, fecha y ubicacion.
3. Sube imagen opcional del evento.
4. Publica el evento.

### 5.3 Editar evento
Si eres creador o moderador (segun permisos):
1. Abre el evento.
2. Selecciona editar.
3. Guarda cambios.

## 6. Chat
### 6.1 Chat de evento
- Puedes enviar mensajes de texto.
- Puedes adjuntar imagenes si la opcion esta habilitada.
- El historial se carga al entrar al chat.

### 6.2 Mensajes privados
- Puedes abrir conversacion con otro usuario.
- Puedes revisar historial y mensajes no leidos.

## 7. Perfil de usuario
En el perfil puedes:
- Editar datos personales.
- Cambiar foto de perfil.
- Gestionar preferencias de categorias.
- Actualizar contrasena.

## 8. Recomendaciones y guardados
La app genera recomendaciones segun actividad y categorias.
Tambien puedes:
- Guardar eventos de interes.
- Consultar listas personales de eventos/rutas guardadas.

## 9. Notificaciones
Recibiras notificaciones dentro de la app para eventos relevantes y actividad relacionada.

## 10. Solucion de problemas frecuentes
### No cargan eventos
- Comprueba conexion a Internet.
- Reinicia la app.
- Verifica que el backend este operativo.

### No puedo iniciar sesion
- Verifica correo/contrasena.
- Intenta recuperar contrasena.
- Confirma que Firebase este disponible.

### No puedo subir imagen
- Acepta permisos de galeria/camara en el dispositivo.
- Verifica tamano y formato de imagen.

## 11. Privacidad y uso responsable
- No compartas datos sensibles en chats.
- Respeta normas de convivencia.
- Reporta contenido inapropiado al equipo de moderacion.
